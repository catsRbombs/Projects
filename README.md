This repository functions as a portafolio to showcase all my previous projects where I used different data structures to create programs that showcase experience with languages such as:

-Node.js
-MySQL
-Python
-C++
-Java
-NASM Assembly
-HTML
-CSS
-JavaScript

A small summary of each project is:
auth-prototype: A prototype for a microsoft-styled authenticator application for the web. It uses firebase and javascript, and permits registering and loging in accounts with thorough security.
Battleships and Buildings: A small game using threads and Java where you build different tools to create recourses on a grid and destroy your opponents recourses while trying to build your own.
Character Battles: A kind of pokemon-style game built with threads and Java where you choose some characters and get abilities for them, and you fight other players using your characters which battle theirs.
Contacts App: A simple Python app that serves the purpose of saving different contacts using their phone number, and lets you create groups and categories.
IslandSurvivorz: A Sandbox Java game built on Netbeans where you place down a couple of characters and humans, and can give them different recourses and use their different abilities to survive through the different days where natural disasters and dangerous animals attack them.
SortingComparator: A simple C app with an interface built with Glade that allows you to compare the efficiency of some of the most common sorting algorithms.
Super Tic Tac Toe: A tic-tac-toe on every tic-tac-toe tile, where you and another player or against an automatic algorithm play together to try to win 3 games in a line within the biggest grid. Built on NASM Assembly.
Tournament Maker: A simple Python application where you can create soccer tournaments and store diferent teams, using typical soccer tourny rules.
Tree Sandbox Game: An app built on c++ that works as a factory that uses different data structures such as trees and connected nodes with different amounts of connections to create humans, let them die, and search for the data of up to hundreds of thousands data points.
Tricount: A simple TUI finance app that runs on linux built on NASM assembly.
